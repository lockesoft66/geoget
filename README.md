Geoget is a Tool which gives you an easy way to test the actual Release of PCGEOS (https://github.com/bluewaysw/pcgeos/releases/tag/CI-latest-issue-829) in combination with the latest Basebox Release (https://github.com/bluewaysw/pcgeos-basebox/releases/tag/CI-latest-issue-13).
To use Geoget you have two ways.
1. ) The easy Way is to use the Files you can find at Releases. Download the File wich matter to your computer and start it in a Terminal / shell. Add the name of the directory which wpild contain your Installation as an parameter.

geoget-win64.exe geogetbx

as example for an 64 Bit Windows and the directory geogetbx which get created in your User-directory.
After this you can start the Launcher in your new created directory.

2. ) If you want to use the realy newest Version from Geoget, for testing and there is a new version, you can use Git to clone this project. To do this start a shell in your user directory or start a shell and change to your userdirectrory. Type  git clone https://github.com/huberthuckevoll/geoget. Git creates a directory geoget in your userdirectory in this you can find the executables. Rest of Procedere is like described in 1. ).
